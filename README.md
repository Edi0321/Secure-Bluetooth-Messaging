# Bluetooth Messaging App

Split it in 3 different Parts that Each Memeber Implemented


-Bluetooh Implemented By Edvin Rastoder

-Cryptography Implemented By Noah Ruderman

-Routing Implemented By Seyed Nima Morsali (Nick)


Requirements
- python 3.11


Run the following commands to complete setup
```
python3 -m pip install -r requirements.txt  
```

to run the program on Server Side in order:  
`python3 crypto.py`
`python3 Server.py`

to run the program on Server Side in order:  
`python3 Client.py`

The web-page can be loaded directly by opening index.html in a browser, Then Server side needs to start messaging first
